Game Rule:
1. Each Player choose a starting point at their corresponding zones. Both Player can't see their opponent's point until both player has place their points.

2. During each Player's turn, the Player can put a point onto an empty tile adjacent to a point that Player owns.

3. Player can set up a cannon on a point that Player owns.

4. Player can fire an already set up cannon, which destroy the first opponent tile that is in front of the cannon. Fired cannon can't be fire again.

5. Player can clean up a destroyed tile adjecent to a point that player owns.

Winning Condition:
1. If one Player make a point onto the opponent's zone, that Player wins.

2. If one Player can't move, that Player loses.
